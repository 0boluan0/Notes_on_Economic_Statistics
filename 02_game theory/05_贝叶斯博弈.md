# 贝叶斯博弈

## 五元素

相较于普通的博弈的三个要素，贝叶斯博弈有五个元素，分别是

### player

这里的这个player没有把不同类型的人分开，类型的分类会归后面一项来区分

### type

同一个player的不同回报，每种回报认定为一个type。

### action

每个人能够选择的行为。

特别的，对于有不同type的player，不同type的可选范围一定要相同，如果存在不一样的可选范围，那么仅靠看对方的选择，另一方就能得到关于该方的type的信息，违背了基本假定。

### belief

一方对另一方的type的分布的预设，仅反应这一方的一厢情愿，不是对方的真的type的分布情况。

### payoff

不仅与行动策略组合（strategy profile）有关，也和类型组合有关，就是题目那个表格里的数字。

##  将五元素转化为3要素

### player

新的player的定义从自然人扩展到了每个自然人的每个类型单算为一个player。

### strategy

由于player的变化，strategy也相应扩展为每个自然人的每个类型的策略选择

### payoff

不变

# Global Game

## 分析

global game相较于Bayesian game 自己的扰动和对方的扰动密切相关
![[Pasted image 20240409105306.png]]
知道了自己的也就大概知道对方的。

## eg：政府工作招标

两家公司去前期勘查，得出一项工程的大概成本，这个成本就是我的type，当知道这个成本的时候我就知道了对方估计的成本的大概区间。

所以我的策略应给是比别人低，但只能低一丁点。

* 一级密封拍卖：所有人出价，密封到一个密封袋里，投标，价低者得。
* 二级密封拍卖：基本和一级一样，只不过最后胜者得到的是比他高一级的投标者的出价。

但是，无论是一级还是二级拍卖，获胜者一般相同，且政府的期望成本一样

## 辨析：Global Game 和Bayesian Game

关键在belief，Bayesian 与自己的类型 无关，Global和自己的类型有关。

## 基本信息

### 混合策略的纳什均衡

![[Pasted image 20240411195529.png]]

混合策略的纳什均衡：选择进攻的概率为c/v。当成本越大时越想进攻。

## 变体：correlated types

使进攻成功的收益是固定的，为10，而进攻的成本不确定。

现在两方都派人去打探，一般可能获得真实的c值，一般可能性获得真实值+-1的错误值。其中+1。-1各占一半。一人打探为真，另一人只能打探为假。

![[Pasted image 20240411200217.png]]

### 分析

>以我收到5的信号为例

分情况讨论，一种情况我收到的是真实值，另一种情况我收到的是假的值。讨论后有：无论哪种情况都是：对手1/2取4，对手1/2取6。

那么此时，由于上面分析过的，如果对手有大于c/v的概率选择attach，我就应该选择attack。

### 结论


我得到的信号为0时，我应进攻。

我收到信号为0，则我对信号的期望是0，那此时我的对手的信号为-1，或者1。当对手收到的信号是-1，他一定会进攻，因为进攻严格占优策略。后边就不用讨论了，因为对手收到-1的概率是1/2。即对手选择进攻的概率大于1/2。

由上方的逻辑类推，如果收到的信号为1，2，3，4我都应该进攻。

反着推，当我收到的信号为10时，一定跑。

由此类推9.8.7.6都要跑。

5是一个中间值，比较模糊。进攻和不进攻无差异。

==不确定性却带来了结果的确定性==

为什么会造成这样的结果？

策略有互补性，信号也有协同的机制。二者共同决定了在某些参数范围内会存在一定的确定性。
